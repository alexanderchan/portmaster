# portmaster

## 0.2.0

### Minor Changes

- Add `env` command to output ports in .env format

  - New `portmaster env` command outputs all assigned ports in `.env` format (e.g., `DEV_PORT=3142`)
  - Supports `--prefix` option to add prefix to variable names
  - Supports `--no-uppercase` option to keep variable names lowercase
  - Updated README with Docker Compose integration guide
  - Updated README with npm scripts examples and fallback patterns
  - Added philosophy section explaining portmaster as a helper tool
